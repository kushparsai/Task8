# Simple Blog Layout Using Bootstrap 5
A fully responsive blog page built with Bootstrap 5 CDN, featuring a navbar, blog cards, and a footer with icons.
This website is ideal for beginners learning Bootstrap layout fundamentals.

## Objective
Create a simple, mobile-friendly blog page using Bootstrap 5 components such as:
Navbar
Cards
Grid system
Footer
Utility classes

## Tools Used
VS Code
Bootstrap 5 CDN
Browser (Chrome recommended)

## How to Run
Download or clone the project folder.
Open the folder in VS Code.
Open index.html in your browser.
That’s it — your blog layout will load.

## Features
✔ Responsive Navbar
A fully collapsible Bootstrap navbar with brand name and links.

✔ Blog Cards Section
3 cards arranged in a grid

Each card contains an image, title, description, and a "Read More" button
Uses Bootstrap grid (row, col-md-4)

✔ Footer with Social Icons
Includes Facebook, Instagram, and Twitter icons using Bootstrap Icons CDN.
